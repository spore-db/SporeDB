# Demo Dataset Attribution

The demo fermentation data included with SporeDB is inspired by experiments
conducted at the **Advanced Biofuels and Bioproducts Process Development Unit
(ABPDU)** at Lawrence Berkeley National Laboratory.

## Source

ABPDU Fermentation Bioreactor Sample Data
https://abpdu.lbl.gov/resources/fermentation-bioreactor-sample-data/

## Citation

If you use this data in publications, please cite:

> Advanced Biofuels and Bioproducts Process Development Unit (ABPDU),
> Lawrence Berkeley National Laboratory, Berkeley, CA.
> https://abpdu.lbl.gov/

## Data Description

Three Rhodosporidium toruloides fermentation runs on different carbon sources:
- **abpdu_rt_glucose_001.csv** -- Glucose feedstock
- **abpdu_rt_xylose_001.csv** -- Xylose feedstock
- **abpdu_rt_hydrolysate_001.csv** -- Lignocellulosic hydrolysate feedstock

## Note

If the original ABPDU dataset was unavailable at packaging time, these files
contain synthetic data that mimics realistic R. toruloides bioreactor behavior
(sigmoidal OD600 growth, pH drift, dissolved oxygen consumption patterns).
The synthetic data is suitable for demonstrating SporeDB features but should
not be used for scientific analysis.
